# دانلودر زیرنویس SubSource

دانلود زیرنویس برای فایل‌های ویدیویی از طریق API
[SubSource](https://subsource.net). از **۱۹ زبان** پشتیبان می‌کند.
بهترین نتیجه را به عنوان `video.{lang}.srt` در کنار ویدیو ذخیره می‌کند
و تعداد مشخصی فایل ZIP پشتیبان نگه می‌دارد.

نسخه‌ی اصلی به زبان **Rust** (CLI + GUI) است. نسخه‌های پایتون قدیمی‌ترند.

## دریافت باینری

از صفحه [Releases](https://github.com/saeedrss/subsourceCLI/releases)
فایل مناسب سیستم‌عامل خود را دانلود کنید:

- `sub-rs-x86_64-windows.exe` — ویندوز
- `sub-rs-x86_64-linux` — لینوکس
- `sub-rs-x86_64-macos` — مک (اینتل)
- `sub-rs-aarch64-macos` — مک (Apple Silicon)

## ساختن از سورس

```bash
cd sub-rs
cargo build --release
./target/release/sub-rs --directory "/path/to/videos"
```

## استفاده (CLI)

```bash
# اسکن پوشه و دانلود زیرنویس فارسی (پیش‌فرض)
./sub-rs --directory "G:\1080\MyShow"

# دانلود زیرنویس انگلیسی
./sub-rs --directory "G:\1080\MyShow" --lang en

# اجرای رابط گرافیکی
./sub-rs --gui
```

### گزینه‌ها

| آرگومان | پیش‌فرض | توضیحات |
|---|---|---|
| `--directory` | `.` | مسیر پوشه برای اسکن فایل‌های ویدیویی |
| `--top` | `5` | تعداد زیرنویس‌های پشتیبان برای نگه‌داری |
| `--api-key` | متغیر محیطی `SUBSOURCE_API_KEY` | کلید API |
| `-l, --lang` | `fa` | کد زبان مقصد (fa, en, ar, fr, de, ...) |
| `--proxy` | — | آدرس پروکسی |
| `--dry-run` | — | فقط نمایش عملیات بدون دانلود واقعی |
| `--no-recursive` | — | فقط پوشه ریشه را اسکن کن |
| `--gui` | — | اجرای رابط گرافیکی |

### ساختار خروجی

```
video.mkv
video.fa.srt          ← بهترین نتیجه (مثلاً برای زبان فارسی)
sub/
  video_sub1_*.zip    ← بهترین نتیجه (پشتیبان ZIP)
  video_sub2_*.zip    ← گزینه‌های جایگزین (تا سقف --top)
```

پسوند فایل زیرنویس با توجه به زبان انتخاب شده تغییر می‌کند:
- `--lang fa` ← `video.fa.srt`
- `--lang en` ← `video.en.srt`
- `--lang ar` ← `video.ar.srt`

## رابط گرافیکی (GUI)

```bash
./sub-rs --gui
```

ویژگی‌های GUI:
- **انتخاب زبان زیرنویس** — منوی کشویی با ۱۹ زبان
- **تغییر زبان رابط** — دکمه EN/FA برای جابجایی بین فارسی و انگلیسی
- **پروکسی با قابلیت فعال/غیرفعال‌سازی** — چک‌باکس برای روشن/خاموش کردن پروکسی
- **درباره** — پنجره با اطلاعات نسخه و لینک گیت‌هاب
- **نمایش جزئیات** — لاگ به‌تفکیک هر فایل
- **Dry Run** — تست بدون دانلود واقعی

## زبان‌های پشتیبانی شده

| کد | زبان | کد | زبان |
|---|---|---|---|
| `fa` | فارسی | `en` | انگلیسی |
| `ar` | عربی | `fr` | فرانسوی |
| `de` | آلمانی | `es` | اسپانیایی |
| `ru` | روسی | `tr` | ترکی |
| `ur` | اردو | `ku` | کردی |
| `it` | ایتالیایی | `pt` | پرتغالی |
| `nl` | هلندی | `zh` | چینی |
| `hi` | هندی | `bn` | بنگالی |
| `ps` | پشتو | `az` | آذربایجانی |
| `ckb` | سورانی | | |

## ویژگی‌ها

- باینری Rust تک‌فایل (~10 مگابایت)، بدون نیاز به Python یا runtime
- GUI با egui (کراس‌پلتفرم: ویندوز، مک، لینوکس)
- تشخیص الگوی `S01E01`، تطبیق زیرنویس با فصل و قسمت
- ۱۹ زبان مختلف برای دانلود زیرنویس
- پشتیبانی از پروکسی با قابلیت فعال/غیرفعال‌سازی
- ذخیره‌سازی پشتیبان ZIP از بهترین نتایج
- تأخیر ۱ ثانیه‌ای بین درخواست‌های API
- پنجره About با لینک گیت‌هاب و نسخه
