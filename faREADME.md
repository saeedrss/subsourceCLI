# دانلودر زیرنویس فارسی

دانلود زیرنویس فارسی برای فایل‌های ویدیویی از طریق API
[SubSource](https://subsource.net). بهترین نتیجه را به عنوان
`video.fa.srt` در کنار ویدیو ذخیره می‌کند و تعداد مشخصی
فایل ZIP پشتیبان نگه می‌دارد.

## نیازمندی‌ها

- پایتون ۳.۸ به بالا
- کتابخانه `requests` (به `requirements.txt` مراجعه کنید)

## نصب

```bash
pip install -r requirements.txt
```

## استفاده

```bash
python script.py --directory "G:\1080\MyShow" --top 5
```

### گزینه‌ها

| آرگومان         | پیش‌فرض                    | توضیحات                              |
|-----------------|----------------------------|--------------------------------------|
| `--directory`   | `.`                        | مسیر پوشه برای اسکن فایل‌های ویدیویی |
| `--top`         | `5`                        | تعداد زیرنویس‌های پشتیبان برای نگه‌داری |
| `--api-key`     | متغیر محیطی `SUBSOURCE_API_KEY` | کلید API (در صورت نبود از کلید پیش‌فرض استفاده می‌کند) |
| `--dry-run`     | —                          | فقط نمایش عملیات بدون دانلود واقعی   |
| `--no-recursive`| —                          | فقط پوشه ریشه را اسکن کن (بدون زیرپوشه) |
| `--proxy`       | `None`                     | آدرس پروکسی (مثلاً `http://127.0.0.1:8080`) |

### ساختار خروجی

```
video.mkv
video.fa.srt          ← بهترین نتیجه (استخراج و تغییر نام داده شده)
sub/
  video_sub1_*.zip    ← بهترین نتیجه (پشتیبان ZIP)
  video_sub2_*.zip    ← گزینه‌های جایگزین (تا سقف --top)
```

## ویژگی‌ها

- تشخیص الگوی `S01E01` در نام فایل‌ها، تطبیق زیرنویس با فصل و قسمت
- در صورت خالی بودن عنوان استخراج شده، از نام پوشه به عنوان جستجو استفاده می‌کند
- تأخیر ۱ ثانیه‌ای بین درخواست‌های API برای رعایت محدودیت نرخ
- پشتیبانی از پروکسی برای دور زدن تحریم یا فیلتر
- حذف کاراکترهای غیر ASCII از نام فایل‌ها برای سازگاری با کنسول ویندوز
